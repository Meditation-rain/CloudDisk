# CloudDisk 测试清单

本文档用于第 11 阶段验收。建议每次提交前在模拟器中完整跑一遍。

## 运行前准备

- 使用 Xcode 打开 `CloudDisk.xcodeproj`。
- 选择 `CloudDisk` scheme。
- 选择 iPhone 模拟器。
- 如遇旧数据干扰，可删除模拟器中的 CloudDisk App 后重新运行。

## 基础启动

- [ ] App 可以正常启动。
- [ ] 底部显示 `网盘` 和 `文件` 两个 Tab。
- [ ] `网盘` 页面可以上下滚动。
- [ ] `文件` 页面可以展示根目录文件。

## 数据初始化

- [ ] 首次启动后，文件列表来自 mock JSON。
- [ ] 根目录显示 `学习资料`、`视频收藏`、`项目说明.txt`。
- [ ] 文件行显示类型、大小、时间。
- [ ] 首页显示最近转存列表。

## 文件夹浏览

- [ ] 点击 `学习资料` 可进入子目录。
- [ ] 子目录显示 `SwiftUI 笔记.txt`、`SQLite 设计.txt`、`归档`。
- [ ] 点击 `返回上一级` 可返回根目录。
- [ ] 空文件夹显示空状态。

## txt 阅读器

- [ ] 点击 `项目说明.txt` 可进入阅读器。
- [ ] 阅读器显示文本内容。
- [ ] 左滑进入下一页。
- [ ] 右滑返回上一页。
- [ ] 页码显示正确。
- [ ] 第一页禁用上一页按钮。
- [ ] 最后一页禁用下一页按钮。

## 视频播放器

- [ ] 进入 `视频收藏`。
- [ ] 点击 `产品演示.mp4` 可进入系统播放器页面。
- [ ] 页面标题显示视频文件名。

## 最近浏览

- [ ] 打开任意 txt 文件。
- [ ] 返回 `网盘` Tab。
- [ ] 最近浏览列表出现刚打开的文件。
- [ ] 再打开另一个文件后，最近浏览排序更新。

## 文件管理

- [ ] 点击文件页右上角 `+` 可打开系统文件选择器。
- [ ] 选择本地文件后，文件复制到 App 沙盒并显示在当前目录。
- [ ] 上传文件后，首页最近转存列表更新。
- [ ] 上传 txt 后可以进入阅读器。
- [ ] 上传 mp4 后可以进入系统播放器。
- [ ] 长按文件出现操作菜单。
- [ ] 选择 `重命名`，输入新名称后列表刷新。
- [ ] 空文件名不能保存，并显示错误提示。
- [ ] 选择 `移动`，可移动到根目录或其他文件夹。
- [ ] 移动文件夹时不能移动到自身。
- [ ] 选择 `删除`，确认后文件从列表消失。
- [ ] 删除文件夹时，其子文件也被移除。
- [ ] 操作完成后显示完成提示。

## 分享与 DeepLink

- [ ] 长按文件，选择 `分享链接`。
- [ ] App 弹出分享结果页。
- [ ] 链接格式为 `clouddisk://share?id=share_xxx`。
- [ ] 链接中不包含文件名、路径等明文信息。
- [ ] 链接会复制到剪贴板。
- [ ] 在模拟器 Safari 输入链接后，App 可被拉起。
- [ ] App 拉起后展示分享文件页。

## 异步与体验

- [ ] 页面加载时显示 loading。
- [ ] 刷新文件列表不会卡住页面。
- [ ] 重命名、移动、删除时不会阻塞界面。
- [ ] 数据库读写不会导致明显卡顿。
- [ ] 错误信息可以在页面中看到。

## 已知边界

- [ ] 分享记录为本机本地数据库记录，不支持跨设备访问。
- [ ] 分享记录为本机本地数据库记录，不支持跨设备访问。
